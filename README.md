# Coz
Türkçe sözdizimli programlama dili

## Konsola Yazma
```ruby
"merhaba" yaz
```

## Konsoldan Okuma
```ruby
a: oku
```

## Dosyaya Yazma
```ruby
"deneme.txt" ye "merhaba" yaz
```

## Dosyadan Okuma
```ruby
a: "deneme.txt" den oku
```

## Pencereyle Gösterme
```ruby
pencereye "merhaba" yaz
```

## Pencereden Okuma
```ruby
a: pencereden oku
```

## Değişken Atama
```ruby
a: "merhaba"
a yaz
```

```ruby
a: 5
a: 5+10*3
```

## Metin Birleştirme

```ruby
a: "Ben " . "Abdullah"
"kur: " . 1.25 yaz
```

## Koşullar
```ruby
a > 10 ise "10dan büyük" yaz
```

```ruby
a = 35 ise {
  "35miş" yaz
  a: 10
}
```

## Döngüler
```ruby
a kere "merhaba" yaz
```

```ruby
a: 0
10 kere {
  a: a+1
  a yaz
}
```
